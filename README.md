[fse] is [fs] then-ified, extended, and dependency-free. It is also;

- improved `fs.mkdir` and `fs.writeFile` methods allowing recursive creation, and;
- new `fs.copydir`, `fs.copyFile`, `fs.rmdir`, and `fs.touchFile` methods.
